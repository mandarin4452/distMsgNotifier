# [kor] Disaster Message Email Notifier

재난 문자 알림을 웹 상에서 조회 & 이메일로 최근 내역을 확인할 수 있는 웹 어플리케이션입니다.

## Requirements

``` pip install flask flask_sqlalchemy apscheduler```
